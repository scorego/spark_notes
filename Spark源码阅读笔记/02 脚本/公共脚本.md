有一些公共处理逻辑抽象为了独立的脚本，如：

| 脚本                    | 说明                                          |
| ----------------------- | --------------------------------------------- |
| `sbin/spark-config.sh`  | 初始化环境变量`SPARK_CONF_DIR` / `PYTHONPATH` |
| `bin/load-spark-env.sh` | 初始化环境变量`SPARK_SCALA_VERSION`           |
| `conf/spark-env.sh`     | 加载用户自定义环境变量                        |
